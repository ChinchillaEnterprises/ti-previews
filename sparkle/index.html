TI Sparkle Pill · Blue + Yellow Stars

Navy blue bubble with yellow sparkle stars. Stars glow + twinkle when you type. Sweep highlight runs while the AI is processing. Matches the BeonInfinity / TI rebrand color (navy #101B3A) paired with the original Beon-yellow sparkles you wanted back.

Three states

Idle is the resting outline. Active fires the moment a character lands in the input — fills navy, stars turn yellow and twinkle. Processing layers a sweeping highlight on top of active while the AI is thinking.

Idle Empty input, cursor not focused
✦✦
Active · on type User has typed at least one character
✦✦
Processing After submit, while AI generates the answer
✦✦

Try it · live input

Click into the bar and start typing. Pill activates on first character. Hit return to switch into processing for ~3s, then it relaxes back to idle.

Spec for Bridge

Drop-in CSS for the sparkle pill component. Activation logic is two class toggles: .active on input change, .processing on submit.

CSS
.sparkle-pill {
  display: inline-flex; align-items: center; gap: 4px;
  height: 36px; padding: 0 14px;
  border-radius: 999px;
  border: 1.5px solid #101B3A;
  background: transparent;
  transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
  position: relative; overflow: hidden;
}
.sparkle-pill .star {
  font-size: 16px; color: #101B3A;
  transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
}

.sparkle-pill.active {
  background: #101B3A;
  border-color: #101B3A;
  box-shadow: 0 0 18px rgba(255, 239, 0, 0.40);
}
.sparkle-pill.active .star {
  color: #FFEF00;
  text-shadow: 0 0 8px rgba(255, 239, 0, 0.6);
  animation: starTwinkle 1.4s ease-in-out infinite;
}
.sparkle-pill.active .star:nth-child(2) {
  animation-delay: 0.7s;
}

@keyframes starTwinkle {
  0%, 100% { transform: scale(1);    opacity: 1;   }
  50%      { transform: scale(1.25); opacity: 0.75; }
}

.sparkle-pill.processing::after {
  content: ''; position: absolute; inset: 0;
  background: linear-gradient(90deg, transparent, rgba(255,239,0,0.35), transparent);
  animation: sweep 1.6s linear infinite;
}
@keyframes sweep { 0% { left: -100%; } 100% { left: 100%; } }

@media (prefers-reduced-motion: reduce) {
  .sparkle-pill.active .star,
  .sparkle-pill.processing::after { animation: none !important; }
}
